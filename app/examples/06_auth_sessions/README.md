# 06 - Session Authentication

What you learn:
- Session-based login with Flask-Login
- Password hashing
- Protecting routes

## Install

```
pip install Flask Flask-Login Flask-SQLAlchemy psycopg2-binary
```

## Configure

Set `DATABASE_URL`:

```
DATABASE_URL=postgresql+psycopg2://USER:PASSWORD@localhost:5432/flask_learning
```
