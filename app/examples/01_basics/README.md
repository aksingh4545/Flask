# 01 - Basics (Functions, Decorators, App Setup)

What you learn:
- Flask app creation and configuration
- Functions as view handlers
- Custom decorators with `functools.wraps`
- Request/response basics and JSON

## Run

```
set FLASK_APP=app.py
flask run
```

## Try

- `GET /`
- `GET /greet?name=Ravi`
- `GET /sum/3/5`
- `GET /headers`
